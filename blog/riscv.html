RV32I

I will start with the simplest ISA and assembly and then build up my way to the harware implementation. RV32I (base integer instruction set) will be used as the base for the implementation. The details are in chapter 2 of the specification manual for the unprivileged ISA: specification manual
IEU is the execution and write back to memory in the fetch-decode-execute. In RISC-V, Harts (hardware threads) are the synonyms of cores, each with its own registers, IEUs and ALUs. By convention a program can be multithreded in several harts (if it was written to do so).
RV32I has 32 registers, each 31 (x1.. x31) general purpose registers + x0 (alwayszero) and the pc.

Instructions

RV32I has 47 instructions, I will just basically be copy-pasting here from RVALP Releases which is also a nice and a readable resource of the RFV32I assembly ISA.

usage type description detailed description
addi rd, rs1, imm i add immediate rd ← rs1 + imm, pc ← pc + 4
lw rd, imm(rs1) i load word rd ← m[rs1 + imm], pc ← pc + 4
sw rs2, imm(rs1) s store word m[rs1 + imm] ← rs2, pc ← pc + 4
beq rs1, rs2, imm b branch if equal if rs1 = rs2 then pc ← pc + imm else pc ← pc + 4
